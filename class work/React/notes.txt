---------activity 2------
1.what is comp and types of comp,where we use comp in react ,with 2 example
->A component is a small, reusable piece of UI (like a button, header, form). It helps break the UI into manageable parts.
types of components:
1.Functional components: These are simple JavaScript functions that return JSX. They are easier to
write and understand. Example:
function Greeting() {
  return <h1>Hello, World!</h1>;
}
2.Class components: These are ES6 classes that extend React.Component and have a render method that returns JSX. They can manage state and lifecycle methods. Example:
class Greeting extends React.Component {
  render() {
    return <h1>Hello, World!</h1>;
  }
}


2.what is props,types,why we use props,with 2 example
->Props (short for properties) are a way to pass data from a parent component to a child component in React. They are read-only and allow components to be dynamic and reusable.
Types of props:
1. Primitive props: These are simple data types like strings, numbers, and booleans. Example:
function Greeting(props) {
  return <h1>Hello, {props.name}!</h1>;
}
2. Object props: These are more complex data types like arrays and objects. Example:
function UserProfile(props) {
  return (
    <div>
      <h1>{props.user.name}</h1>
      <p>Age: {props.user.age}</p>
    </div>
  );
}
