// difference b/w package.json and package-lock.json
// ->package.json is a file that contains the metadata of the project and the dependencies of the project. It is used to manage the dependencies of the project and to run scripts.
// ->package-lock.json is a file that contains the exact version of the dependencies that are installed in the project. It is used to ensure that the same version of the dependencies are installed in every environment.

//difference b/w dependency and devDependency
// ->dependency is a package that is required for the project to run. It is used in the production environment.
// ->devDependency is a package that is required for the development of the project. It is used in the development environment and is not included in the production environment.

//http request methods

// POST: used to send data to the server.
// PUT: used to update data on the server.
// DELETE: used to delete data from the server.
// PATCH: used to update a part of the data on the server.
